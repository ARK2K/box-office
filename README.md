# Box Office

A responsive React app to search movies and actors using OMDb API, with Firebase authentication and cloud favorites storage.

## Features

✅ **Search** - Search movies and actors from OMDb database
✅ **Authentication** - Email/Password and Google Sign-In with Firebase
✅ **Cloud Favorites** - Save favorites to Firebase Firestore
✅ **Mobile-First** - Fully responsive design (mobile, tablet, desktop)
✅ **Real-Time Sync** - Favorites sync instantly across devices

## Tech Stack

- **React 18** - UI Framework
- **Vite** - Build tool
- **Tailwind CSS v4** - Styling
- **Firebase** - Authentication & Database
- **OMDb API** - Movie/Actor data
- **GitHub Pages** - Deployment
